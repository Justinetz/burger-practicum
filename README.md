**Рекомендация:**
Для самостоятельного запуска и доработки проекта нужно
- склонировать код локально и открыть в VSCode или GitBash
- установить зависимости `npm i`
- запустить npm run dev

По окончании доработок необходимо выполнить команду `npm run lint` для выявления ошибок.
